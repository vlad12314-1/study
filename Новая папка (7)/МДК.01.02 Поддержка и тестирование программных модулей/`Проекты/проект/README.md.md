# Игра "Угадай число"

## Описание
Простое desktop-приложение-игра "Угадай число". Компьютер загадывает случайное число, а пользователь пытается его угадать, получая подсказки "больше/меньше".

## Функциональность
- Генерация случайного числа в заданном диапазоне
- Проверка введенного пользователем числа
- Информативные подсказки (больше/меньше)
- Подсчет количества попыток
- Возможность начать новую игру
- Ввод числа через текстовое поле или клавишу Enter

## Требования
- Python 3.7+
- PyQt5
